Digital Archives - HIST-GA-1011

Hello! Welcome to the Digital Archives (HIST-GA-1011) course site, containing all things related to the class. Here, you’ll find the course details, week-by-week schedule listing assigned readings, activities and assignment due dates, and descriptions for both assignments and weekly activities. We will use this site alongside Brightspace (used primarily for assignment submissions) throughout the semester.

Why GitHub? This site is built using GitHub Pages, a platform that allows for writing, versioning and hosting documents, while also making it easy to publish a functional website for free.

If GitHub is new to you, don’t worry! You won’t need to use it directly for this course. For now, just know that it helps keep class materials accessible and organized. Building this site on GitHub also reflects a conscious decision to reduce reliance on platforms like Google and Brightspace. This ensures that course content is is openly available and not tied to institutional accounts, and in a readable format (markdown) that would make it easier to transition this content to another platform if need be.
